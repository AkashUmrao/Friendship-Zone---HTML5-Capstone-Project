# Friendship Zone - HTML5 Project

Friendship Zone is a web application that allows individuals to connect with their friends.  

## Pages
- **Homepage.html** → Main home page
- **FriendsList.html** → Shows details of friends
- **RegistrationForm.html** → Enroll at Friendship Zone
- **RegistrationConfirmation.html** → User registration success
- **RegistrationConfirmation_admin.html** → Admin registration success

## Features
- Simple and clean HTML5 design
- Easy navigation
- Friendship connection platform
